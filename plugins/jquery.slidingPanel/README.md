# jQuery Menu Level

Multi-level navigation


## Features

* IE8+ compliant


## Installation

### 1. Join plugin

```html
<script src="assets/js/plugins/jquery.slidingPanel.js"></script>
```

### 2. JavaScript

```js
$('#sliding-panel').slidingPanel({
  trigger: $('#sliding-panel-trigger'),
  wrapper: $('#site')
});
```

### 3. Options

Options | Type  | Description                    | Default
--------|-------|--------------------------------|--------
lorem   | Array | Images paths                   | null
