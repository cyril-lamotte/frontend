    box-sizing: false
  )

    Description

    Susy configuration map

    Example

    Usage

    1/3 - 2/3

  @include container;

  > .grid-item {
    @include span(4 of 12);

    +  .grid-item {
      @include span(8 of 12 last);
    }
  }

}

    $override: (gutters: 0);
@include span($override $col-count-12 of 12);

    breakpoints

    $breakpoints: (
  'xs': 480px,
  'sm': 768px,
  'md': 992px,
  'lg': $max-site-width
);

    Description

    Breakpoints map - Based on Bootstrap reference

    • Phones portait < 480px
    • Phones < 768px
    • Tablets >= 768px
    • Desktops >= 992px
    • Large desktops >= $max-site-width

    Used by

    • [mixin] mq-to (max-width)
    • [mixin] mq-from (min-width)
    • [mixin] mq-from-to (min-width & max-width)

    Links

    • Bootstrap breakpoints reference
