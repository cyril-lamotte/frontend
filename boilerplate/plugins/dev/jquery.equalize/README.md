# jQuery Equalize

Equalize blocks height which have the same offsetTop.


## Features

* IE8+ compliant
* Class equalized is added after equalizing


## Installation

### 1. Join plugin

```html
<script src="js/jquery.equalize.js"></script>
```

### 2. JavaScript

```js
$(window).load(function() {

  $('.blocks').equalize();

});
```
