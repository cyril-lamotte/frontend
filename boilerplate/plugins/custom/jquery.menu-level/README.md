# jQuery Menu Level

Multi-level navigation


## Features

* IE8+ compliant


## Installation

### 1. Join plugin

```html
<script src="assets/js/plugins/jquery.menu-level.js"></script>
```

### 2. JavaScript

```js
$('.menu-level-wrapper').menuLevel();
```

### 3. Options

Options | Type   | Description                    | Default
--------|--------|--------------------------------|--------
prefix  | string | Classes prefix                 | 'menu-level-'
