# jQuery selectAll

Select All checkboxes in a container


## Installation

### 1. Join plugin

```html
<script src="assets/js/plugins/jquery.selectAll.js"></script>
```

### 2. JavaScript

```js
$('#select-all').selectAll();
```

### 3. Options

Options | Type         | Description                    | Default
--------|--------------|--------------------------------|--------
wrapper | CSS selector | Wrapper parent element         | 'form'
